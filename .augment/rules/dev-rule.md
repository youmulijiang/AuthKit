- 开发者是youmulijiang
- 代码遵循高内聚低耦合
- 开发时注意线程安全，禁止在 UI 线程执行耗时 diff 运算
- 不能随便添加其他功能，必须按照开发者的指令编写相关功能的代码
- 项目采用测试驱动开发，在编写重要逻辑代码时，先编写测试，当测试通过时在编写代码
- 开发者没有允许下，不能git commit message
- 插件的入口是AuthKit
- 命名遵循语义化
- 在包中，可以用包名作为尾缀，比如在model包中可以命名为DiffModel
- 采用设计模式进行编写，在编写UI时，采用建造模式编写ui组件，采用观察者模式进行组件通信
